
# Angular Installation

## Installation 

- [Check Angular Releases](https://github.com/angular/angular/releases)

```bash
# uninstallation
npm uninstall -g @angular/cli

# Installation
npm install -g @angular/cli
npm install -g @angular/cli@18.2.10

# Creation
ng new angular-starter

```
---


## SSR & PWA

```bash
# PWA Installation
ng add @angular/pwa

# SSR installation
ng add @angular/ssr

```
